Acesta este un site creat "pe graba" si o solutie de front-end.

Dorinta noastra a fost sa cream un site web unde orice cetatean sa gaseasca informatiile de care are nevoie atunci cand interactioneaza cu statul roman.

Misiunea lui este de a reduce birocratia si de a combina mai multe site-uri ale unor institutii de stat intr-unul singur. De asemenea, se doreste a fi o sursa clara si sigura de informatii pentru cetatenii straini ce au legatura cu tara noastra.
